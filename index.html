01 About

A bit about me and how I like to build.

I'm Jay Shelke, a full-stack developer based in Pune, India. I like building complete products end to end rather than isolated demos — each of my three projects covers a different real-world problem: real-time messaging, a transactional payments flow, and secure, per-user data handling.

I try to make thoughtful choices in what I build — using rate limiters, cleaning up related data on deletion, using database aggregation where it makes sense, and keeping authentication secure. These are the details I care about getting right.

Currently looking for a full-stack or backend-leaning developer role or internship where I can keep learning and contribute to a product-first team.

A Full-Stack Stock Trading Platform
Live demo GitHub

Full-stack MERN trading simulator with transactional order execution, so wallet debits, order creation, and holdings updates always succeed or roll back together.
